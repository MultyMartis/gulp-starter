Комплектация

В стандартной комплектации

  • Купель из кедра
  • Обручи из нержавеющей стали
  • Слив воды
  • Лестница (2-х ступенчатая)
  • Сиденья внутри купели (по всему периметру)
  • Инструкция по эксплуатации

Дополнительное оборудование

  • Печь (внутренняя или наружная)
  • Крышка для купели (пластиковая / тентовая)
  • Подиум для купели
  • Хромотерапия (подсветка воды)
  • Гидромассаж (система аэромассажа)
  • Полочка для напитков
  • Фильтровальная установка для очистки воды
  • Электрический водонагреватель (если нет печи)

В стандартной комплектации

  • Купель из кедра
  • Слив
  • Обручи из нержавеющей стали
  • Лестница (приставная или навесная, в зависимости от модели)
  • Сиденья из кедра внутри
  • Инструкция по эксплуатации

Дополнительное оборудование

  • Подиум под купель
  • Крышка (деревянная или термокрышка)
  • Оборудование (печь, фильтрация, подсветка, гидро- и аэромассаж)
  • Аксессуары (весла, столики, подстаканники, полотенцедержатели)
  • Средства по уходу за купелью
  • Сервисное обслуживание
