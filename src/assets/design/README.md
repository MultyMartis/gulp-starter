# Design materials

This folder stores design materials for frontend implementation.

Each page has its own folder (for example `home`, `about`).

Inside each page folder:

- **original** — raw design files (source exports, Figma links file, etc.)
- **exports** — exported desktop / mobile / tablet screens (PNG, PDF, JPG)
- **images** — raster assets extracted for build use
- **icons** — SVG, logo, and icon assets
- **notes** — comments and implementation notes

AI must inspect this folder before design-based implementation.
